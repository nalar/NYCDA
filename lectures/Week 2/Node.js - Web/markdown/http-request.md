# Anatomy of a URL

```
<protocol>://<subdomain>.<domain name>.<top level domain>:<port number>/route/to/resource

https://mail.google.com/mail/u/0/#inbox
https://www.google.com/#q=boop
http://74.125.224.72/#q=beep

```
